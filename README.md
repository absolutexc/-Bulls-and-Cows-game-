# -Bulls-and-Cows-game-
Игра "Быки и коровы" на языке С++

Как играть?
Правила игры:
1) Загадывается 4-значное число
2) Цифры в числе не повторяются
3) Цифры в числе от 0 до 9
4) Число МОЖЕТ начинаться с цифры 0
5) 'Быки' - правильные цифры на правильных местах
6) 'Коровы' - правильные цифры на неправильных местах
7) Какие именно цифры угаданы - не сообщается
Пример: загаданное число - 7890, попытка - 7609
Вывод: кол-во быков: 1, кол-во коров: 2

Как запустить игру? 
1 вариант: (работает без сред разработки, запускается на компьютере)
Запустить файл cpp.exe

2 вариант: (работает только в средах разработки таких как Clion, VS Community, VS Code и т.д.)
1) Загрузить файлы с репозитория себе на компьютер
2) Переместить их в один проект
3) Запустить файл main.cpp или functions.cpp
